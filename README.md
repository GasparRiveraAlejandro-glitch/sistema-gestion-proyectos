# 📊 Sistema de Gestión de Proyectos

Sistema completo de gestión de proyectos con autenticación, CRUD de proyectos, tareas, dashboard con gráficos y colaboración.

## 🛠️ Tecnologías

- **Backend:** Node.js, Express, MongoDB, JWT
- **Frontend:** React, Recharts, Axios

## ✨ Características

- ✅ Registro y login de usuarios
- ✅ CRUD de proyectos
- ✅ CRUD de tareas con estados (pendiente, en progreso, revisión, completada)
- ✅ Dashboard con gráficos estadísticos
- ✅ Colaboración (miembros en proyectos)
